Polished, localized recipe discovery for Polish-speaking food enthusiasts. Responsive, image-rich UI with intuitive navigation, contextual back-tracking, and focused search keeps users immersed in everyday cooking inspiration.

React React Router Tailewind Vite

Project Overview

This project delivers a polished recipe discovery web application tailored for Polish-speaking food enthusiasts. Visitors can browse curated recipe collections, perform targeted searches, and dive into detailed preparation guides. A responsive layout paired with engaging imagery positions the platform as both inspirational and practical for everyday cooking. The experience emphasizes intuitive navigation, contextual back-tracking, and localized content to keep users immersed in the culinary journey.

Project Description

Gotujto reimagines a traditional recipe portal through the lens of modern frontend engineering. The objectives were to provide a frictionless browsing experience, encourage exploration across categories, and support power users with fast search and filtering. Target users range from home cooks seeking dinner inspiration to food lovers eager to save favorites and share recommendations. Unique aspects include a fully localized interface (labels, copy, and CTAs in Polish), a dynamic “Przepisy” catalog that reflects popularity and freshness, and an “O nas” microsite built from modular components for scalability.

Technology Stack

Implementation & Process

Development began with restructuring legacy markup into reusable React components and establishing page-level routing for /, /przepisy, /kategoria/:slug, /przepisy/:id, /favourites, and /o-nas. Context hooks centralize data fetching and favorites management, while presentational components encapsulate styling and behavior.

Navigational UX required special handling: the header highlights active routes, the hero CTA and sidebar respect localized slugs, and recipe detail pages remember the originating view to provide meaningful back actions. Implementing Polish-specific slugs introduced encoding/decoding challenges when sharing category context via router state; helper logic now normalizes and prettifies category labels. The mobile menu implements animated collapse behavior and icon toggling, keeping accessibility attributes accurate.

Results & Learnings

The refactor produced a cohesive, multi-page SPA with fully localized copy and consistent branding across sections. Popular and newest recipe grids now respond to live metadata (views, publication timestamps), reinforcing relevance for users. The work deepened expertise in crafting resilient navigation flows with React Router, managing shared state without heavy dependencies, and translating static HTML designs into scalable component architectures. Localization revealed nuances—such as preserving diacritics across URLs and UI—that will inform future internationalized projects.
