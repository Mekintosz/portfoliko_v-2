Personal Blog

A publication-ready blog that balances storytelling, code snippets, and accessibility—crafted for fast iteration and content ownership.

Next.js MDX Vercel Prism

The blog serves as a living knowledge base for engineering notes and long-form essays. I focused on authoring ergonomics, themeable typography, and optimizations that keep Time to Interactive under one second even with media-heavy content.

Technology Stack

Built on Next.js with hybrid rendering for fast page loads and SEO-friendly routes. Content is authored in MDX and synced from a Git-based CMS. Syntax highlighting uses Prism with a custom theme, while Vercel handles zero-config deployments and preview builds.

Key Features

Authoring Workflow

MDX components simplify embedding callouts, interactive demos, and code sandboxes without leaving the markdown flow.

Accessible Typography

A modular scale, tuned contrast ratios, and reading-progress indicators help readers stay oriented on long articles.

Newsletter Integration

Subscribers sync to ConvertKit via serverless functions, providing analytics and welcome sequences right after publish.

Visual Showcase

Development Process

Typography studies informed the baseline styles before coding began. Components were documented in Storybook, and performance budgets were enforced with automated Lighthouse CI. Accessibility checks using axe DevTools run on every pull request.

Results & Impact

The blog consistently scores 95+ on Lighthouse for performance and accessibility. Organic traffic doubled within three months thanks to fast loads and structured metadata tailored for social sharing.
