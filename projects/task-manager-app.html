Task Manager App

A productivity companion that keeps teams aligned with shared workspaces, smart reminders, and progress insights across every device.

Vue Firebase Pinia Tailwind CSS

Built for remote teams juggling multiple projects, the app makes light work of deadlines and accountability. I created the real-time collaboration features, ensured offline resilience, and designed a component library that adapts from mobile to desktop seamlessly.

Technology Stack

Vue 3's composition API provides a clean architectural backbone, while Pinia manages shared state between boards, tasks, and user preferences. Firebase powers authentication, Firestore real-time data, and Cloud Functions for scheduled reminders. Tailwind's utility classes were converted into the portfolio's vanilla CSS token system for consistency.

Key Features

Shared Workspaces

Organize projects into boards with drag-and-drop prioritization and visibility controls that keep cross-functional teams in sync.

Smart Reminder Engine

Custom reminders leverage Cloud Functions to nudge owners when due dates slip, factoring in working hours and task dependencies.

Progress Insights

Built-in analytics highlight velocity, blockers, and task distribution so leads can rebalance workload quickly.

Visual Showcase

Development Process

We kicked off with user interviews to define the jobs-to-be-done, then ran weekly usability tests to validate flows. Component-first development in Storybook accelerated iteration, and E2E tests with Cypress ensure real-time sync remains reliable under load.

Results & Impact

Teams using the app report a 40% reduction in status meetings and a noticeable boost in sprint completion rates. Real-time presence and automated reminders keep everyone aligned without micromanaging.
