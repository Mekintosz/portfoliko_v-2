Gotujto.pl

A feature-rich online store built to deliver a seamless shopping experience—from browsing curated collections to a frictionless checkout flow.

React React Router Tailewind Vite

The platform reimagines product discovery for a specialty retail brand. I was responsible for architecture, component implementation, and creating a reliable state layer that keeps the UI responsive while synchronizing inventory, user sessions, and personalized recommendations in real time.

Technology Stack

The project uses a modular React/TypeScript setup bundled with Vite. Redux Toolkit manages complex cart and authentication flows, while react-query handles data fetching and caching. Styling is powered by a custom component library built with vanilla CSS tokens for predictable theming.

Key Features

Personalized Product Discovery

Dynamic home and category pages adjust to browsing history, highlight trending items, and surface cross-sell bundles that increase average order value.

Seamless Checkout Journey

A multi-step checkout wizard with address validation, payment status feedback, and guest checkout support keeps conversion high on every device.

Storefront Management Dashboard

Admin tools let merchandisers manage inventory, launch campaigns, and monitor funnel metrics with real-time dashboards built on the same component system.

Visual Showcase

Development Process

The build followed an iterative delivery cycle. We began with UX prototypes in Figma, established a shared design token system, and then delivered components incrementally. Automated tests cover the core cart logic and payment hand-offs, while Lighthouse audits guard performance budgets every sprint.

Results & Impact

Launch metrics captured a 32% lift in conversion rate and a 24% decrease in cart abandonment. The composable architecture now supports regional storefronts without rewriting the experience, unlocking rapid iteration for future campaigns.
